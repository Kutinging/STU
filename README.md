<h3>1.使用方法</h3>
1-1.把程式碼新增為書簽  
1-2.開啟獨立的評量網頁(如圖)  
<font color="red">※如何開啟:登入校務資訊系統後找到<strong>教學成效問卷填答</strong>並點<strong>在新分頁中開啟連結</strong></font>  
![image](https://github.com/Kutinging/STU/raw/master/imgs/CoStuAns.png)  <br/>

1-3.點選要評量的課程後直接點選書籤將自動選擇選項並自動提交表單
